Description

Currently a prototype, Fire Run is a 2D Platformer. It was made for the UE4 summer game jam 2018 : "That escalated quickly" theme. The level is created using procedural generation. You create spreadable fire as you move

Play as the flame "flamy" who just kissed a match in a very big and dry forest... Search for an escape from your own flames. But be carefull to water !

How to run the game

Download and extract the .zip archive. Run ue4Jam.exe.

Please note that the game requires a XInput compatible Gamepad Controller, a 64bit version of windows and a screen resolution of at least 1280x720.
